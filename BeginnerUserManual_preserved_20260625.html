Telepathy Beginner User Manual

Overview

Telepathy Beginner is a two-person practice and experiment app. One participant acts as the sender and the other acts as the receiver. The app supports three difficulty levels, shared pair identity, server-side history, performance reporting, and optional administrative tools.

Home Screen

The main launcher shows two expandable brown panels:

  • I Will Be the Sender
  • I Will Be the Receiver

Each panel can show the current level for the active receiver-sender pair. A newly formed pair defaults to Level 1.

Entering Names

Each participant should enter:

  • Your name
  • The exact name of the other participant

The pair identity is role-sensitive. A receiver-sender arrangement is different from the same two people with the roles reversed.

Starting a Session

  • The receiver may press GO first.
  • The sender may press GO first.
  • Once both are online with a matching receiver-sender pair, the shared session becomes active.

If two people accidentally choose the same role for the same pair, the app warns that the role is already occupied and returns that participant to the home screen.

Difficulty Levels

Level 1

  • Targets are limited to the 1-cone center target and the four 3-cone corner targets.
  • After the countdown, the receiver chooses between One cone and Many cones.
  • After confidence is entered, the app shows a 5-target feedback grid.

Level 2

  • Targets are limited to the same 5-target subset used in Level 1.
  • The receiver chooses directly from the 5-target grid.
  • The sender sees the mirrored grid with the correct target highlighted.

Level 3

  • The full 9-target task is used.
  • This is the only level where the optional second receiver choice can apply.

Receiver-Side Settings

Open Options > Settings to manage receiver-side options for the current device.

  • Allow a second choice as well as a first choice applies only to Level 3.
  • Download CSV data filename can be used when downloading CSV data.

The old in-trial settings screen is no longer the normal control path. The gear icon now returns to launcher Settings.

Changing Difficulty

Open Options > Change difficulty level of the task.

  • Only the receiver of the pair can change the active difficulty.
  • The level is stored as a shared setting for that receiver-sender pair.

Options Screen

The Options screen currently provides:

  • View performance report
  • Change difficulty level of the task
  • Settings
  • User manual

Performance Reports

Performance history is now read from server-side per-pair records.

  • Choose View performance report.
  • The app opens a Report Definition screen first.
  • Select the desired receiver-sender pair.
  • Press GO to open the report.

The report shows the history stored on the server for that selected pair.

Interruptions, Aborts, and Timeouts

  • The Home icon can be used during a trial if a participant needs to stop.
  • If one participant exits, the partner is told that the trial was ended and is returned to the home flow.
  • If a timeout occurs, both sides are informed and the interrupted status is recorded in trial history when appropriate.

Administrative Tools

Authorized administrative tools are available only after the hidden admin code is entered through the Settings flow.

Current administrative functions include:

  • Enable or disable debugging
  • Clear the debug log
  • List all users, roles, partners, and trial counts
  • Analyze disk usage

Data Storage

Completed receiver-side trials are written to the server-side pair history. The system uses a public/private split:

  • public browser-facing app files stay in the web directory
  • private runtime and history files stay outside the public web root

This allows reports and administrative review to be performed from other devices while keeping operational data out of the public website directory.

Installing to a Phone Home Screen

iPhone / Safari

  • Open the website in Safari.
  • Tap the Share button.
  • Choose Add to Home Screen.
  • Tap Add.

Android

  • Open the site in your phone browser.
  • Use the browser install or add-to-home-screen option if available.
  • If needed, locate the installed app in the app drawer and drag it onto the home screen.

Notes

  • Use exact consistent names for the same receiver-sender pair over time.
  • For serious experiment review, rely on the server-side history and reports rather than browser-local history.
  • If you are unsure which level or settings are active, check the launcher panels and the receiver-side Settings screen.
